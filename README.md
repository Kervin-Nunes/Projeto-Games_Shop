# Games Shop 🎮

Landing page de uma loja fictícia de games desenvolvida com **HTML5 e CSS3 puro**. A página apresenta seções de apresentação da loja, exibição das marcas parceiras, formulário de contato e links para redes sociais.

## ✨ Funcionalidades

- **Navbar** com âncoras para as seções da página
- **Seção "Sobre a Loja"** com identidade visual da marca
- **Logos das marcas** Nintendo, Xbox e PlayStation
- **Formulário de contato** para mensagens
- **Links para redes sociais** (Instagram, Facebook e YouTube)
- **Rodapé** com endereço físico e copyright

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## 🚀 Como rodar

Basta abrir o `index.html` diretamente no navegador.
